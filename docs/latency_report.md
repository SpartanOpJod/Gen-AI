# Latency Benchmark Report

## Metrics

- Retrieval p50: median retrieval latency over the benchmark.
- Retrieval p95: 95th-percentile retrieval latency.
- End-to-end p95: 95th-percentile latency from query to answer generation.

## Output

The benchmark results are written to [evaluation/latency_results.json](../evaluation/latency_results.json).
